# 🏢 Mapa de Disponibilidade de Unidades

Plataforma web interativa para visualização e acompanhamento da disponibilidade de unidades em empreendimentos imobiliários.

Ideal para **lançamentos imobiliários**, permitindo que clientes e equipes comerciais acompanhem, em tempo real, quais apartamentos estão disponíveis, reservados ou vendidos.
---

## 💡 Problema

Durante lançamentos imobiliários, o controle de unidades disponíveis costuma ser:

* pouco visual
* descentralizado
* dependente de planilhas ou comunicação manual

Isso dificulta tanto a experiência do cliente quanto a eficiência do time comercial.

---

## ✅ Solução

Este projeto oferece uma interface visual e intuitiva onde:

* cada unidade é representada graficamente
* o status é atualizado de forma clara
* usuários conseguem identificar rapidamente a disponibilidade

---

## 🧩 Funcionalidades

* 🗺️ Visualização interativa do empreendimento
* 🏢 Representação individual das unidades
* 🎨 Diferenciação visual por status:

  * Disponível
  * Reservado
  * Vendido
* ⚡ Atualização rápida de status
* 👀 Experiência simples e intuitiva para clientes

---

## 🛠️ Tecnologias Utilizadas

* Frontend: HTML, CSS, JavaScript 
* Visualização: Manipulação de DOM / SVG
* Backend: Node, Python
* Dados: JSON / API
